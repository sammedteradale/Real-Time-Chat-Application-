the project REAL-TIME-CHAT-APPLICATION, The main objective of this project was to build a bridge between two people so they can talk to each other online instantly. instead of normals websites which take more time to refresh, mainly prefered for live experience, it involves the development of a fullstack project, REAL-TIME-CHAT-APPLICATION utilized using socket.io for the backend and react for frontend,
goals are :: instant delivery , private rooms , modern interface like you can see who sent the message at what time.

Tools i used : React.js > used for to build the chat screen.
               Node.js & Express : this is are used in the middle. to intract user                with server.
               Socket.io : used to enable real-time.
               NPM : i used this to download all the extra parts i needed.


> Building this chat app was a great way to see how modern internet                communication works.It taught me that web development isn't just about making things look pretty; it’s about managing how data moves through the air.
